O que é CSS

Cascading Style Sheets (CCS) é usado para formatar yma página da web. Com CSS, você pode controlar a cor, a fonte, o tamaho do texto, o espaçamento entre os elementos como os elementos são posicionados e dispostos, quais imagens de fundo ou cores de fundo devem ser usadas, exibições diferentes para dispositivos e tamanho de tela diferentes e muito mais.

O que é HTML?

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto de junção entre os padrões HyTime e SGML. Hytime é um padrão par a representação estruturada de hipermídia e conteúdo baseado em tempo.

O que é JavaScript?

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da Word Wide Web.
